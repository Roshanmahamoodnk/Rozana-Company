Free GitHub Pages OCR website

Read text from images and PDFs in the browser

This is a fully static OCR web app you can host on GitHub Pages for free. Upload an image or PDF, run OCR client-side, and copy or download the extracted text.

Client-side OCR Image support PDF support No backend Free hosting
100% static Works on GitHub Pages without a server.
Browser OCR Uses Tesseract.js and pdf.js directly in the browser.
Easy share link Once Pages is enabled, anyone can access your site URL.

1. Upload

Drop an image or PDF, or click to choose files.

2. Extract

Text is read directly in the browser, page by page.

3. Export

Copy the output or download it as a text file.
